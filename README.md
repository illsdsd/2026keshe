# CampusLink —— 校园竞赛组队与协作平台

> 哈尔滨工业大学（威海）软件工程综合课程设计 · 第 6 组

一个集 **找队友 → 申请加入 → 组队成功 → 任务协作 → 比赛结束 → 队员互评** 于一体的校园竞赛组队与协作闭环平台。

## 技术栈

| 层 | 技术 |
| --- | --- |
| 前端 | Vue 3 + Vite + Element Plus + Pinia + Vue Router + Axios |
| 后端 | Spring Boot + MyBatis Plus + Spring Security + JWT |
| 数据库 | MySQL 8.0 |
| 缓存（加分项） | Redis |
| 部署 | Nginx 反向代理 + 可选 Docker Compose |

## 功能模块

1. 用户模块：注册登录、JWT 鉴权、个人主页、技能标签管理
2. 竞赛模块：竞赛信息发布与浏览、按类型 / 时间筛选
3. 队伍模块（核心）：创建队伍、浏览、多维筛选、招募岗位
4. 申请加入模块：申请、审核、状态流转
5. 队内协作模块：任务管理、看板拖拽、进度可视化
6. 公告模块：公告发布、未读提醒
7. 消息模块：站内消息中心、已读 / 未读
8. 评价系统：匿名互评、信誉分汇总

## 快速开始

### 数据库
```bash
mysql -uroot -p < sql/campuslink.sql
```

### 后端
```bash
cd backend
mvn spring-boot:run   # 默认端口 8080，上下文 /api
```

### 前端
```bash
cd frontend
npm install
npm run dev           # 默认端口 5173
```
