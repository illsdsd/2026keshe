# CampusLink —— 校园竞赛组队与协作平台

> 哈尔滨工业大学（威海）软件工程综合课程设计 · 第 6 组 · v2.0.0

一个集 **找队友 → 申请加入 → 组队成功 → 任务协作 → 比赛结束 → 队员互评** 于一体的校园竞赛组队与协作闭环平台。

🚀 **v2 扩容**：从 8 个模块 / 50 接口 / 19 页面 / 12 张表 扩展到 **11 个模块 / 116 接口 / 44 页面 / 32 张表**。详见 [v2 技术方案](20260627-【CampusLink】课设系统扩容技术方案.md)。

## 技术栈

| 层 | 技术 |
| --- | --- |
| 前端 | Vue 3 + Vite + Element Plus + Pinia + Vue Router + Axios + **ECharts** |
| 后端 | Spring Boot 3 + MyBatis Plus + Spring Security + JWT + **AOP 操作日志** + **Spring Task** + **Apache POI** |
| 数据库 | MySQL 8.0 |
| 文件存储 | 本地磁盘（`~/campuslink/uploads`，可切换 OSS） |
| 邮件 | mock 模式（控制台打印验证码），可切真实 SMTP |
| 部署 | Nginx 反向代理 + 可选 Docker Compose |

## 功能模块（v2）

### v1 基础
1. 用户模块、2. 竞赛模块、3. 队伍模块、4. 申请加入、5. 队内协作、6. 公告、7. 消息、8. 评价

### v2 扩容
9. **文件存储模块**：通用上传/下载/分片/我的文件中心
10. **管理员后台**：用户/队伍/举报管理、数据大盘、平台公告、技能标签
11. **数据统计与可视化**：个人雷达图、队伍报表、平台趋势、Excel 导出

### v2 主要新增能力
- 账号安全：找回密码、修改密码、账号锁定、登录日志、隐私设置
- 用户中心：项目作品集、证书、收藏（竞赛 / 队伍）
- 赛事：报名 → 审核 → 附件 / 资讯 / 排行榜
- 队伍：文件库、动态墙 + 评论 + 点赞、副队长、归档、招募置顶、黑名单
- 协作：任务优先级 / 标签 / 评论、子任务 + 状态联动、工时填报 + Excel 导出、任务模板、甘特图
- 评价：评价回复、举报、信誉分明细 + 排行榜
- 底层：全局操作日志切面、5 个定时任务、数据字典、模拟邮件、统一返回码扩充

## 快速开始

### 数据库
```bash
# v1 主脚本（首次部署）
mysql -uroot -p < sql/campuslink.sql
# v2 增量脚本（必须在 v1 之后执行）
mysql -uroot -p campuslink < sql/v2_extension.sql
```

### 后端
```bash
cd backend
mvn spring-boot:run   # 默认端口 8080，上下文 /api
# 文件存储目录默认 ${user.home}/campuslink/uploads，可在 application.yml 修改
# 邮件验证码默认 mock，控制台打印；切真实 SMTP 设 campuslink.mail.mock=false
```

### 前端
```bash
cd frontend
npm install
npm run dev           # 默认端口 5173
```
