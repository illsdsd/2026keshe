# CampusLink 页面原型说明

共约 19 个页面，以下为主要页面布局（文字原型）。

## 全局布局
```
┌────────────────────────────────────────────┐
│  Logo CampusLink   竞赛  队伍  我的队伍   🔔3  头像▾ │  顶部导航
├────────────────────────────────────────────┤
│                                            │
│                 页面内容区                  │
│                                            │
└────────────────────────────────────────────┘
```

## 1. 登录 / 注册页
```
        ┌──────────────────────┐
        │     CampusLink       │
        │  ┌────────────────┐  │
        │  │ 账号            │  │
        │  │ 密码            │  │
        │  └────────────────┘  │
        │   [   登录   ]       │
        │   没有账号？去注册     │
        └──────────────────────┘
```

## 2. 个人主页
```
┌──────────┐  昵称 朱展硕   信誉分 ⭐4.8
│  头像    │  软件学院 · 软件工程 · 2023级
└──────────┘  技能：[Java][Vue][算法] + 添加
个人简介 / 项目经历 ........................
[编辑资料]
```

## 3. 竞赛列表页
```
筛选：[类型▾] [关键词____] [搜索]
┌────────────────────────────┐
│ 蓝桥杯   程序设计  截止 06-30 │ 详情
│ 数模国赛 数学建模  截止 09-15 │ 详情
└────────────────────────────┘
            < 1 2 3 >
```

## 4. 队伍列表页（核心）
```
筛选：[竞赛▾][学院▾][年级▾][技能▾][关键词][搜索]
┌──────────────────────────────────────┐
│ 代码先锋队   关联：蓝桥杯   3/4 人      │
│ 简介：冲刺国一……                       │
│ 缺口：Vue开发×1  UI设计×1   [查看][申请] │
└──────────────────────────────────────┘
            < 1 2 3 >
```

## 5. 队伍详情页
```
代码先锋队  [招募中]  3/4
关联竞赛：蓝桥杯       队长：Alice
简介：……
招募岗位：Vue开发 0/1 ｜ UI设计 1/1
成员：Alice(队长) Bob Carol
[申请加入]  (队长可见：[编辑][解散][审核申请])
```

## 6. 创建队伍页
```
队伍名称 [____________]
关联竞赛 [下拉选择▾]
所属学院 [____________]
总人数   [ 4 ]
队伍简介 [____________]
招募岗位 [岗位__][人数_] + 添加一行
            [ 创建队伍 ]
```
